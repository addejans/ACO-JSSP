# Legacy prototype scripts

This folder contains the original 2017 ACO-JSSP prototype scripts. They are preserved for historical reference.

New work should start from `src/aco_jssp.py`, which contains the modern Python 3 implementation, generated instances, local search, tests, and CLI support.

Known legacy issues include Python 2 syntax, deprecated Plotly APIs, hard-coded Plotly credentials, heavy global state, and no automated tests.
